# 状态栏修复器 StatusBar Fixer

SillyTavern 扩展插件，自动检测并修复 AI 回复中损坏或缺失的状态栏格式。

## 安装方法

在 SillyTavern 扩展管理中输入：
https://github.com/orangeyxq2006-hue/statusbar-fixer

## 配置说明

1. 启用插件后进入扩展设置面板
2. 填写你的 API Endpoint、API Key、模型名称
3. 把预设或世界书里的状态栏格式定义粘贴进「格式模板」框
4. 保存设置

## 使用方法

点击输入框旁边的 🔧 按钮，插件会自动检测并修复最后一条AI回复的状态栏格式，弹出预览确认后替换。
